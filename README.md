# ALX Listing App

The ALX Listing App is a Next.js-based Airbnb clone project for displaying property listings.

## Project Structure

- components/common: Reusable UI components (Card, Button)
- interfaces: TypeScript interfaces for props and data types
- constants: Reusable constants like API URLs and text
- public/assets: Static files such as images and icons

## Running the Project

1. Install dependencies:
   npm install
2. Run development server:
   npm run dev
3. Open browser:
   http://localhost:3000
